# Glossary

## Disposition Paths
- **RESELL**: Sent to B2B auctions or B2C storefronts. Highest value recovery.
- **REPAIR**: Sent to refurbishing lab for part replacements (e.g., screen) before RESELL.
- **PARTS**: Cannibalized for functioning internal components.
- **RECYCLE**: E-waste processing for materials.
- **QUARANTINE**: Immediate segregation due to physical hazards (e.g., swollen battery).

## Terms
- **RVP (Recovery Value Percentage)**: `Estimated Recovery Value / Base Market Value`. If RVP drops below a configured threshold, the device is usually cannibalized or recycled.
- **Lowest-Grade Rule**: A deterministic business rule stating that the final device grade cannot exceed its lowest individual component grade. (e.g. Cosmetic A + Functional C = Final Grade C).
- **Decision Intelligence**: Using multi-agent AI to optimize the recovery value of a device based on real-time market data, rather than static routing tables.
